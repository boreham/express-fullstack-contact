Приложение форма обратной связи REST API Express, TypeScript, Dotenv (имя, фамилия, email, телефон, прикрепить вложение, текст).
Структура проекта:
```
backend/
├── src/
│   ├── controllers/
|   |   └── contactController.ts        
│   ├── repositories/
│   │   └── contactRepository.ts
|   ├── routes/
│   │   └── contactRoutes.ts
|   ├── services/
│   │   └── contactService.ts
|   ├── utils/
│   │   └── validation.ts
│   └── app.ts
├── .env
├── package.json
└── tsconfig.json
```
